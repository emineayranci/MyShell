# MyShell

Projede kendi linux shell programımızı yazmamız istendi. Bu kapsamda bu shell programı işlem topla, işlem çıkar, tekrar, cat, clear, exit gibi komutları vardır. Projede bulunan c dosyalarını derlediğimiz zaman aşağıdaki işlemeleri yapmalıdır.

islem topla komutu 2 sayı alır ve bunları toplayarak sonucu döndürür. \n
islem cikar komutu 2 sayı alır ve bunları çıkararak sonucu döndürür. \n
tekrar komutu bir string bir de integer bir ifade alır ve int sayısı kadar o stringi yazar.
cat komutu yanına yazılanı tekrar yazar.
clear komutu terminal ekranını temizler
exit komutu ise programdan çıkar
